----------------------
 Install Instructions
----------------------
1. Unzip 'nsync.zip' to your desktop.
2. Look on your desktop and run 'setup.exe'.
3. Follow the installation wizard to complete the install.
4. Once done, a shortcut will be created on your desktop.
5. Run 'nsync.exe' and enjoy the software!
